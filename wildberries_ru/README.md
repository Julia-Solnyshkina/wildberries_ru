# Wildberries Parser

Парсер товаров с сайта Wildberries по произвольному поисковому запросу.

## 📌 Описание

Этот проект — Scrapy-парсер, который собирает каталог товаров с Wildberries по заданному поисковому запросу.

По умолчанию в константах установлен запрос:  
**"пальто из натуральной шерсти"**, но его можно заменить на любой другой.

## 📦 Что собирается

Парсер извлекает следующие данные о товарах:

• Ссылка на товар
• Артикул
• Название
• Цена (основная)
• Описание
• Ссылки на изображения через запятую
• Все характеристики с сохранением их структуры
• Название селлера
• Ссылка на селлера
• Размеры товара через запятую 
• Остатки по товару (число)
• Рейтинг
• Количество отзывов

## 📌 Обработка вариантов товара

В рамках задания реализован сбор данных по основному варианту товара, представленного в выдаче API Wildberries.

## 📁 Результат

После запуска парсера формируются два XLSX-файла:

1. `all_items.xlsx` — полный каталог товаров по запросу  
2. `filtered_items.xlsx` — отфильтрованные товары с условиями:
   - рейтинг ≥ 4.5  
   - цена ≤ 10000  
   - страна производства: Россия

## ⚙️ Настройка
В проекте используется параметр `dest=-1198055`, который определяет регион выдачи товаров в API Wildberries.

В текущей реализации установлен регион:
**Санкт-Петербург (Россия)**

Этот параметр влияет на:
- доступность товаров
- цены
- наличие на складах
